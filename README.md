# 项目前端网站 服务于个人毕设，仅开放学期末。如果需要。联系作者获取核心代码

## 项目概述

用作个人毕设和课设使用，课设中的人物分析是其中一个功能，这个是整体项目的前端架构

## 主要功能组件

### 核心组件

1. **GeoApiDashboard.vue**：
   - 作为地理信息服务的核心功能模块
   - 集成POI搜索、天气查询、行政区域查询和交通态势查询等功能
   - 可视化展示地理信息与分析数据
   - 集成高德地图显示和交互功能

2. **DeepSeekService.ts**：
   - 封装DeepSeek API调用
   - 支持V3和R1模型选择
   - 提供智能分析功能
   - 支持进度跟踪和流式输出

3. **AnomalyDetectionService.ts**：
   - 提供数据异常检测功能
   - 支持本地和云端检测模式
   - 提供异常分析和报告

### 辅助组件

1. **WeatherVisualization.vue**：
   - 天气数据可视化组件
   - 展示温度、降水、风力和历史天气数据图表
   - 支持响应式布局和数据更新

2. **POIAnalysisComponent.vue**：
   - POI搜索结果分析组件
   - 提供区域分布、评分分布、人流量趋势和场所类型分布图表
   - 智能解析DeepSeek分析结果

3. **MarkdownRenderer.vue**：
   - Markdown渲染组件
   - 支持DeepSeek API返回的Markdown格式内容

4. **ProgressBar.vue**：
   - API调用进度条组件
   - 显示DeepSeek API调用的实时进度和状态

5. **AnomalyDetectionComponent.vue**：
   - 异常检测分析组件
   - 可视化展示异常数据和趋势
   - 集成DeepSeek API进行智能异常分析

## 技术特点

1. **模块化设计**：
   - 将功能拆分为独立组件，提高代码复用性和可维护性
   - 采用服务层分离，降低组件间耦合度

2. **数据可视化**：
   - 使用ECharts提供丰富的数据图表展示
   - 支持多种图表类型：饼图、柱状图、折线图等
   - 响应式设计适应不同屏幕尺寸

3. **AI增强功能**：
   - 集成DeepSeek API提供智能分析
   - 支持天气分析、POI分析、交通状况分析等
   - 异常检测和智能报告生成

4. **高德地图集成**：
   - 地理位置可视化
   - 多种地图类型支持
   - 丰富的标记和信息窗口功能

5. **响应式布局**：
   - 适应桌面和移动设备
   - 可伸缩面板设计
   - 优化的移动端用户体验

## 技术栈

- **前端框架**：Vue 3 + TypeScript
- **UI组件库**：Element Plus
- **数据可视化**：ECharts 5
- **地图服务**：高德地图API
- **AI服务**：DeepSeek API (V3和R1模型)
- **HTTP客户端**：使用原生Fetch API

## 安装运行

```bash
# 安装依赖
npm install

# 启动开发服务器
npm run dev

# 构建生产版本
npm run build
```

## 使用说明

1. **POI搜索**：
   - 输入关键词和城市进行搜索
   - 查看搜索结果列表和智能分析
   - 在地图上展示POI位置

2. **天气查询**：
   - 输入城市名称查询实况天气或天气预报
   - 查看天气图表和智能分析
   - 获取出行建议

3. **行政区域查询**：
   - 查询各级行政区域
   - 在地图上展示区域边界
   - 获取行政区域详细信息

4. **交通态势查询**：
   - 输入矩形区域查询交通状况
   - 查看交通态势分析
   - 在地图上展示交通状况
